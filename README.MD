# mznode-command-start

`mz-node-cli` 启动 server 命令。

## 安装

```bash
npm i -g mznode-command-start
```

`mz-node-cli` 中已添加依赖。

## 使用

遵循 `mz-node-cli` 目录规范。

```bash
Usage: mznode start <options>

Options:

  --server     start nginx and node-server.
  --nginx      start nginx.
  --node       start node-server.
  --build      start build.
```
